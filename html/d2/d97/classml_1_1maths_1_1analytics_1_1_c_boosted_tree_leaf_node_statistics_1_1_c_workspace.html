Detailed Description

The derivatives and row masks objects to use for computations.

DESCRIPTION:
These are heavyweight objects and get passed in to minimise the number of times they need to be allocated. This has the added advantage of keeping the cache warm since the critical path is always working on the derivatives objects stored in this class.
