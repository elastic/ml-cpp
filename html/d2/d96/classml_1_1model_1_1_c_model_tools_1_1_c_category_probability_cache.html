Detailed Description

Wraps up the calculation of less likely probabilities for a multinomial distribution.

DESCRIPTION:
This caches the probabilities for each category, in the multinomial distribution, since they can't be computed independently and for a large number of categories it is very wasteful to repeatedly compute them all.
