Detailed Description

Convert a stream of Base64 characters to bytes.

DESCRIPTION:
Class to convert Base64 characters into bytes using the boost iostreams filter interface.

IMPLEMENTATION DECISIONS:
Uses a boost circular_buffer to cache chunks of up to 4096 bytes from the incoming stream, and another buffer to cache the converted bytes. This class is templated on the stream output type, which must be compatible with boost::iostreams::write. Stream data is copied from the incoming stream, processed locally, then copied into the output stream; this could be improved, at the expense of simplicity, if it proves to be overly slow.
