Detailed Description

Computes the joint probability of seeing a more extreme collection of samples.

DESCRIPTION:
This assumes that the samples are jointly normal with fixed correlation between 0.0 and 1.0, supplied to the constructor, and computes the probability of seeing a more extreme event:

Here are the probabilities supplied to the add function. Note that because it supplied the probabilities, which it interprets as from a standard Gaussian, these can in fact be computed from any distribution.

It is possible to supply a weight with each probability. This can be used to approximately compute the expectation of the joint probability of a collection of samples which are sampled where each sample only appears with some specified frequency. The weights must be non-negative.
