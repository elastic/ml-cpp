template<typename T>
class ml::core::CScopedBoostJsonPoolAllocator< T >

A scoped json memory allocator.

DESCRIPTION:
Control the lifetime of the memory allocator associated with a generic json output writer

IMPLEMENTATION DECISIONS:
Parameterized on json writer type Remove the allocator from the writer and release resources on destruction.
