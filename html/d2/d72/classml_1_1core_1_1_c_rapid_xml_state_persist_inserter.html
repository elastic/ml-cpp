Detailed Description

For persisting state in XML format.

DESCRIPTION:
Concrete implementation of the CStatePersistInserter interface that persists state in XML format.

IMPLEMENTATION DECISIONS:
Directly uses RapidXml to avoid the inefficiency of an intermediate node hierarchy.
