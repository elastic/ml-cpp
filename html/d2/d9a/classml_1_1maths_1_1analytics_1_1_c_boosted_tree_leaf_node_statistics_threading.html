Detailed Description

Utilities for threading CBoostedTreeLeafNodeStatistics code.

DESCRIPTION:
This encapsulates the logic to select the number of threads to use for different operations computing leaf node statistics. Depending on the amount of work to perform using fewer threads (which incur an overhead) can improve throughput. Here, where possible, we estimate the work to perform as a function of the thread overhead and minimize throughput.

IMPLEMENTATION DECISIONS:
The constants in the cost model for each operation were extracted using ordinary least squares regression for a couple of typical environments. This provides a route to calibrate at runtime if we choose. The danger of doing this is that timing statistics could be skewed by temporary environmental factors. We can revisit this if we decide it is important.
