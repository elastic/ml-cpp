Detailed Description

template<typename T>
class ml::core::CConcurrentQueue< T >

A thread safe concurrent queue.

DESCRIPTION:
A thread safe concurrent queue.

This is a simple wrapper around boost::circular_buffer in order to make it thread-safe. As boost circular buffer is bounded and start overwriting when the buffer gets full, the queue blocks if it reaches the capacity, so no message is lost for the price of blocking.
