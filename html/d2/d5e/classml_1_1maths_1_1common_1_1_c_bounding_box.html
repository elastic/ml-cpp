template<typename POINT>
class ml::maths::common::CBoundingBox< POINT >

An axis aligned bounding box.

DESCRIPTION
Holds the bottom left and top right corners and provides various utility functions need by the x-means algorithm.

◆ closerToX()

template<typename POINT >
bool ml::maths::common::CBoundingBox< POINT >::closerToX ( const POINT & x,
const POINT & y ) const
inline

Check if x is everywhere closer to the bounding box than y.

The idea is fairly simple: find the corner which is closest to the hyperplane bisecting the vector from x to y. Note that this plane is boundary of the region where every point is closer to x or y. If the corner is in the region closer to x then the bounding box must necessarily be in this region too and no point can therefore be closer to y than x.
