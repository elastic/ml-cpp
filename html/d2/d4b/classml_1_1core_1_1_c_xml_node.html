Detailed Description

Simple representation of a XML node.

DESCRIPTION:
Simple representation of a XML node.

IMPLEMENTATION DECISIONS:
Only values of a XML node required for XPath returns implemented.

The XML parser is a friend so that it can efficiently populate attributes with minimal copying of data
