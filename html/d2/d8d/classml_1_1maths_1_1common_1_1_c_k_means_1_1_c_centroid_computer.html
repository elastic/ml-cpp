Detailed Description

template<typename POINT>
class ml::maths::common::CKMeans< POINT >::CCentroidComputer

Updates the cluster centres in an iteration of Lloyd's algorithm.

DESCRIPTION:
This is used in a pre-order depth first traversal of the k-d tree of points to efficiently update the cluster centres in one iteration of Lloyd's algorithm. Each point is assigned to its closest centre and the centre placed at the centroid of its assigned points.
