Detailed Description

Functions related to getting process statistics, e.g. memory usage.

DESCRIPTION:
Methods of this class are useful to get process statistics of the running process, e.g. memory usage(RSS).

IMPLEMENTATION DECISIONS:
This is a static class - it's not possible to construct an instance of it.

The implementation is OS specific, not all methods might be available for every OS.
