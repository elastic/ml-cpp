Detailed Description

template<typename POINT>
class ml::maths::analytics::anonymous_namespace{COutliers.cc}::CEnsemble< POINT >

This encapsulates creating a collection of models used for outlier detection.

DESCRIPTION:
A model is defined as one or more algorithm for computing outlier scores, the number of nearest neighbours used to compute the score and a sample of the original data points (possibly) projected onto a random subspace which is searched for neighbours.

The models can be built from a data stream by repeatedly calling addPoint. The evaluation of the outlier score for a point is delagated.
