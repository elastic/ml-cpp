Detailed Description

template<typename T>
class ml::core::CHashing::CMurmurHash2BT< T >

Wrapper for murmur hash to use with basic types.

Warning
This is slower than boost::hash for the types I tested std::size_t, int, std::uint64_t, but does have better distributions.
