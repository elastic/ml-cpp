Detailed Description

Interface for writing the inference model and training data summarization JSON descriptions to a stream which first compresses, base64 encodes and chunks.

IMPLEMENTATION DECISIONS:
This splits large state objects across multiple documents which can be reassembled by CSerializableFromCompressedChunkedJson. Typically this writes documents of the form

"<doc_tag>": {
"doc_num": <number>,
"<payload_tag>": "compressed blob",
["eos": True]
}

where derived types supply doc_tag and payload_tag. This should be used in conjunction with CSerializableFromCompressedChunkedJson to read compressed chunked documents.
