void ml::maths::common::CSampling::multinomialSampleFast ( TDoubleVec & probabilities,
std::size_t n,
TSizeVec & sample,
bool sorted = false )
static

Generate samples from a multinomial distribution with number of trials n and category probabilities probabilities.

This generates a sample from the distribution function

The probabilities are sorted in decreasing order and sample is filled with the counts of each probability in this sorted order. This is the most efficient strategy since often sample will then be much smaller than probabilities and the sampling loop can end early.

Parameters
[in,out]probabilitiesThe category probabilities, which should be normalized.
[in]nThe number of trials.
[out]sampleFilled in with the counts of the each category in the sorted probabilities order. Missing counts are zero.
[in]sortedSet to true if the probabilities are already sorted in descending order.
