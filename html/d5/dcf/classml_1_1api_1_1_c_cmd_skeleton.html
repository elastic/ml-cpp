Detailed Description

Basic implementation of an API command.

DESCRIPTION:
Factors out some boilerplate code.

IMPLEMENTATION DECISIONS:
Input/output streams are held by reference. They must outlive objects of this class, which, in practice, means that the CIoManager object managing them must outlive this object.
