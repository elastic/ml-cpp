Detailed Description

Retrieves data from a single C++ stream.

DESCRIPTION:
Retrieves data from a single C++ stream.

The format must consist of one or more blocks of input in the format returned by Elasticsearch's get API, separated by zero bytes ('\0'). This class will return the same stream for every search. The caller must not close the stream.

Implements the CDataSearcher interface for loading previously persisted data.

IMPLEMENTATION DECISIONS:
It's seemingly ridiculous to return the same stream over and over again, but doing this enables the interface to be used in cases where different streams are returned for each request.
