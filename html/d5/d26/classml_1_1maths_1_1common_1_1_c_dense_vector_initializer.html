Detailed Description

template<typename VECTOR>
class ml::maths::common::CDenseVectorInitializer< VECTOR >

The default type for converting Eigen vectors to our internal vectors.

IMPLEMENTATION DECISIONS:
This type is needed to get Eigen GEMM expressions to play nicely with our vector type constructors. Also, I think it is useful to flag explicitly when a conversion is taking place, the fromDenseVector function plays this role in code where we want a conversion.
