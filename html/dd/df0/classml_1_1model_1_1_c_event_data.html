Detailed Description

The description of the event data corresponding to a single record.

DESCRIPTION:
Records are converted into event data for processing by the model data gatherers. The event data comprises the person which corresponds to either the unique by field, if no over field is present, or the unique over field.

If an over field is present, multiple by fields can be aggregated and so extracted from each record. Therefore, this supports collections of attribute identifiers. Also, more than one metric field can be extracted from the same record, so this supports a collection of metric values.

Finally, each record has a unique time and person, although with different semantics when an over field is and isn't present, so this always holds the time and person identifier.
