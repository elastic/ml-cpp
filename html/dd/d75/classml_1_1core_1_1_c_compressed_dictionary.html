Detailed Description

template<std::size_t N>
class ml::core::CCompressedDictionary< N >

A space efficient representation of a dictionary.

DESCRIPTION:
This implements the concept of a compressed dictionary for arbitrary strings. The idea is to come up with a concise representation of the word strings that have a very low probability of collision. This uses N near independent 64 bit hashes of the words; so even if N is 2 and 260 million different string words are added to the dictionary we expect two different string words to map to the same dictionary word with a probability of less than 1.5e-4.

IMPLMENTATION:
This uses murmur hash 2 behind the scenes because its distributions are good enough and it is very fast to compute.
