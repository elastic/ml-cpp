Detailed Description

Basic wrapper class around pthread_create etc.

DESCRIPTION:
Basic wrapper class around pthread_create etc.

IMPLEMENTATION DECISIONS:
A mutex protects the thread ID variable to prevent race conditions when a thread is started and stopped in quick succession.
