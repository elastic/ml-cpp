Detailed Description

template<typename POINT>
class ml::maths::common::CKMeans< POINT >::CKdTreeNodeData

The data the x-means algorithm needs at each k-d tree node.

DESCRIPTION:
At every node the algorithm needs the axis aligned bounding box of the points in the branch rooted at that node together with their centroid.
