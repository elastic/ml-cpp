Safe Probability Density Function

Unfortunately, boost::math::pdf and boost::math::cdf don't handle values outside of the distribution support very well. By default they throw and if you suppress this behaviour they return 0.0 for the cdf! This wraps up the pdf and cdf calls and does the appropriate checking. The functions are extended to the whole real line in the usual way by treating them as continuous.

Detailed Description

A collection of utility functionality.

DESCRIPTION:
A collection of utility functions primarily intended for use within the maths library.

IMPLEMENTATION DECISIONS:
This class is really just a proxy for a namespace, but a object has been intentionally used to force a single point for the declaration and definition of utility functions within the maths library. As such all member functions should be static and it should be state-less. If your functionality doesn't fit this pattern just make it a nested class.
