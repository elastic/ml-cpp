Detailed Description

For restoring state in JSON format.

DESCRIPTION:
Concrete implementation of the CStateRestoreTraverser interface that restores state in JSON format.

IMPLEMENTATION DECISIONS:
Input is streaming rather than building up an in-memory JSON document.

Unlike the CRapidXmlStatePersistInserter, there is no possibility of including attributes on the root node (because JSON does not have attributes). This may complicate code that needs to be 100% JSON/XML agnostic.
