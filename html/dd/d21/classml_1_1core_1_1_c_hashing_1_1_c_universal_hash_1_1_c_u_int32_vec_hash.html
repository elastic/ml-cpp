Detailed Description

A member of the universal (2-independent) hash family on vectors of integers (Carter and Wegman):

Here, are independent samples from a universal family of hash functions.

Note
This is not the fastest implementation of universal hashing which is the multiply-shift scheme. We can revisit the implementation if this is a bottleneck in practice.
