Detailed Description

Collection of root solving functions.

DESCRIPTION:
This implements a collection of root solving functions as static member functions. In particular, it implements Brent's method and interval bisection.

Maximize the function f on the interval [a, b] This terminates if it has converged on a local maximum or it has run out of iterations. This minimizes minus f using our standard minimization algorithm.

Note that this converges to the unique maximum if there is one. If the function f has several maxima then you could consider using globalMaximize.

[in]toleranceThe convergence threshold ignored if too small so feel free to set to zero.
[in,out]maxIterationsThe maximum number of times f is evaluated and set to the number of times it was evaluated.

Minimize the function f on the interval [a, b] This terminates if it has converged on a local minimum or it has run out of iterations. This implements Brent's method which combines golden section search and quadratic minimization (see for example numerical recipes in C).

Note that this converges to the unique minimum if there is one. If the function f has several minima then you could consider using globalMaximize.

Find the sublevel set of fc the function f on the interval [a, b]. The sublevel set is defined as:

It is assumed that the sublevel set is a closed interval. WARNING this is equivalent to assuming that the function has a unique minimum in the interval [a, b] and the caller should ensure this condition is satisfied.

Parameters
[in]aThe left end of the interval on which to compute the sublevel set of fmax.
[in]bThe right end of the interval on which to compute the sublevel set of fmax.
[in]faThe value of f at a.
[in]fbThe value of f at b.
[in]fThe function to evaluate. This is expected to implement a function signature taking a double and returning a double.
[in]fcThe function value, f(c), for which to compute the sublevel set.
[in]maxIterationsThe maximum number of times f is evaluated.
[out]resultFilled in with the sublevel set of fc if it isn't empty or the point which minimizes f otherwise.
Returns
True if the sublevel set could be computed and false otherwise.
Note
This will evaluate f at most 3 * maxIterations.
