Detailed Description

Reads a file in a thread.

DESCRIPTION:
When the thread is started it makes several attempts to open the file, separated by the specified sleep. This gives some other thread time to create the file. The data is read into a string which can be accessed safely after the reader thread has completed (so this class should not be used to read collosal files that will exhaust memory).

IMPLEMENTATION DECISIONS:
Using core::CThread rather than std::thread to minimise risk as it was factored out of existing code using core::CThread.
