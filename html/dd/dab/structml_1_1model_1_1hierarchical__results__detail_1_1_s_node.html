Detailed Description

A node of the hierarchical results tree.

DESCRIPTION:
A node of the hierarchical results tree. By default we build a tree on top of our simple search results which allows us to obtain partial aggregate results for each object of interest.

For example, if the following searches are run:

  • autodetect count by host
  • autodetect sum(bytes) over host
  • autodetect rare(process) by host

The common field of interest is "host" and we obtain aggregate results for each host as well as an overall aggregate result.

This is used to represent a node of the tree corresponding to this aggregation process.

See also
buildHierarchicalResults for more details.
