Detailed Description

Get the name/location of the current program.

DESCRIPTION:
Get the simple name of the current program and the absolute path of the directory that it was loaded from.

Being able to find out where the program executable is located means other files can be picked up relative to this location.

IMPLEMENTATION DECISIONS:
For the name, just the program name is returned, with no path or extension.
