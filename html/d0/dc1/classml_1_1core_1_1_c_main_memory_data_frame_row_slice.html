Detailed Description

In main memory CDataFrame slice storage.

DESCRIPTION:
This provides maximum speed at the expense of maximum main memory usage. The intention is to provide a speed optimized implementation suitable for small data frames which comfortably fit into memory.

IMPLEMENTATION DECISIONS:
This is basically a wrapper around a single std::vector storing all rows to adapt it for use by the data frame.
