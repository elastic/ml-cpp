Detailed Description

template<typename T>
class ml::model::CBucketQueue< T >

A fixed size queue with the purpose of storing bucket-associated data.

DESCRIPTION:
A queue whose size is fixed to the number of latency buckets plus one, for the latest bucket. The queue allows the storage of bucket-associated data and their retrieval by providing a time. The queue manages the matching of the time to the corresponding bucket.
