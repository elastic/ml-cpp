Detailed Description

template<typename MATRIX>
class ml::maths::common::CDenseMatrixInitializer< MATRIX >

The default type for converting Eigen matrices to our internal symmetric matrices.

IMPLEMENTATION DECISIONS:
This type is needed to get Eigen GEMM expressions to play nicely with our symmetric matrix type constructors. Also, I think it is useful to flag explicitly when a conversion is taking place, the fromDenseMatrix function plays this role in code where we want a conversion.
