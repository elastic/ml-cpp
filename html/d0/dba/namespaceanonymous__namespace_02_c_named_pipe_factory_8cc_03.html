bool anonymous_namespace{CNamedPipeFactory.cc}::ignoreSigPipe ( )

Ignore SIGPIPE, as we don't ever want a process (or even a different thread in the same process) at the other end of one of our named pipes to abruptly terminate our processes. Instead we should handle remote reader death by gracefully reacting to write failures.
