Detailed Description

For persisting state in JSON format.

DESCRIPTION:
Concrete implementation of the CStatePersistInserter interface that persists state in JSON format.

IMPLEMENTATION DECISIONS:
Output is streaming rather than building up an in-memory JSON document.

Unlike the CRapidXmlStatePersistInserter, there is no possibility of including attributes on the root node (because JSON does not have attributes). This may complicate code that needs to be 100% JSON/XML agnostic.
