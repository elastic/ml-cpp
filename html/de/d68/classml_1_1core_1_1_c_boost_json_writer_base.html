Detailed Description

template<typename OUTPUT_STREAM>
class ml::core::CBoostJsonWriterBase< OUTPUT_STREAM >

A Json writer with fixed length allocator pool With utility functions for adding fields to JSON objects.

DESCRIPTION:
Wraps up the code needed to add various types of values to JSON objects.

IMPLEMENTATION DECISIONS:
Empty string fields are not written to the output unless specifically requested.
