static

Perform top-down recursive segmentation with linear models.

The time series is segmented using piecewise linear models in a top down fashion with each break point being chosen to minimise the residual variance. Model selection happens for each candidate segment. This is uses the p-value of the variance the segment explains.

Parameters
[in]valuesThe time series values to segment. These are assumed to be equally spaced in time order.
[in]pValueToSegmentThe maximum p-value of the explained variance to accept a trend segment.
[in]outlierFractionThe proportion of values to treat as outliers. This must be in the range (0.0, 1.0).
[in]maxSegmentsThe maximum number of segments to divide values into.
Returns
The sorted segmentation indices. This includes the start and end indices of values, i.e. 0 and values.size().
