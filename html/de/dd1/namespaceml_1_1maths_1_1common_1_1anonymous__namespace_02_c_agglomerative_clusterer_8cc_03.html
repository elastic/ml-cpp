◆ nnCluster()

template<typename UPDATE >
void ml::maths::common::anonymous_namespace{CAgglomerativeClusterer.cc}::nnCluster ( TDoubleVecVec & distanceMatrix,
UPDATE update,
TDoubleSizeSizePrPrVec & L )

The NN-CHAIN-LINKAGE cluster algorithm due to Murtagh and given in Mullner.

This makes use of the fact that reciprocal nearest neighbours eventually get clustered together by some node of the stepwise dendrogram for a certain class of objective function.

For details see http://arxiv.org/pdf/1109.2378.pdf.

Parameters
[in,out]distanceMatrixthe matrices of distances between the points to cluster.
[in]updateThe distance update function which varies based on the objective function.
[in]LFilled in with the unsorted dendrogram.
Note
This has worst case O(N^2) complexity.
For maximum efficiency modifications are made in place to distanceMatrix.
