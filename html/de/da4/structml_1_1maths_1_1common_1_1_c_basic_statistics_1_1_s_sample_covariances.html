Detailed Description

template<typename POINT>
struct ml::maths::common::CBasicStatistics::SSampleCovariances< POINT >

An accumulator class for vector sample mean and covariances.

DESCRIPTION:
This function object accumulates sample mean and covariances for a set of vector samples passed to its function operator.

Free functions are defined to retrieve the mean vector and covariances to match the behavior of SSampleCentralMoments.

IMPLEMENTATION DECISIONS:
This is templatized to support, for example, float when space is at a premium or long double when accuracy is at a premium. The type T must technically be an infinite field (in the mathematical sense). In particular, any floating point would do, also custom rational or complex types would work provided they overload "*" and "/" and define the necessary constructors; however, integral types won't work.

This uses the same recurrence relations as SSampleCentralMoments so see that class for more information on these.

Template Parameters
POINTThe "floating point" vector type.
