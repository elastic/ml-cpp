Detailed Description

Convert CPrior sub-classes to/from text representations.

DESCRIPTION:
Encapsulate the conversion of arbitrary CPrior sub-classes to/from textual state. In particular, the field name associated with each prior distribution is then in one file.

IMPLEMENTATION DECISIONS:
The serialisation format must be a hierarchical format that supports name/value pairs where the value may be a nested set of name/value pairs. Text format makes it easier to provide backwards/forwards compatibility in the future as the classes evolve.
