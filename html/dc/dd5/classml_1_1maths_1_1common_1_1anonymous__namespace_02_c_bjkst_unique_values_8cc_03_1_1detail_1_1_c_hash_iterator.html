Detailed Description

Random access iterator wrapper for B set iterator.

DESCRIPTION:
This automatically iterates through and extracts the 16 bit hash from the appropriate 8 bit unsigned integer values in the B set for the BJKST algorithm. It assumes that the values in the vector are layed out as follows:

|<-----8 bits---->|<-----8 bits---->|<-----8 bits---->|
|(g(x) >> 8) % 256| g(x) % 256 | zeros(x) |
