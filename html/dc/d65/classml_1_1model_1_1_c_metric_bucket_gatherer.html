Detailed Description

Metric series data gathering class.

DESCRIPTION:
This performs all pre-processing of the data which we model in order to characterize metric time series.

See also
CDataGatherer.

◆ fieldsOfInterest()

const TStrVec & ml::model::CMetricBucketGatherer::fieldsOfInterest ( ) const
overridevirtual

Get the fields for which to gather data.

For individual searches this gets the field which defines the categories whose counts are being analyzed. For population searches this gets the fields identifying the people and person attributes which are being analyzed. An empty string acts like a wild card and matches all records. This is used for analysis which is attribute independent such as total count.

◆ m_FieldNames

TStrVec ml::model::CMetricBucketGatherer::m_FieldNames
private

The names of the fields of interest.

The entries in order are:

  1. The name of the field which identifies people,
  2. For population models only, the name of the field which identifies people's attributes,
  3. The name of zero or more influencing fields,
  4. The name of the field holding the count followed by the field name(s) of the field(s) which hold the statistics themselves, which must (for those that are present) be ordered mean, min, max, sum.
  5. For the API with user defined pre-summarisation, the name of the field which holds the count then the name of the field which holds the statistic value,
  6. Otherwise the name of the field which holds the metric value.
