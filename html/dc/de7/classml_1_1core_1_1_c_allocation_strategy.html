Detailed Description

Container allocation strategy.

DESCRIPTION:
Container allocation strategy. The STL default allocation strategy is to double the memory for containers when they grow beyond their size.

We are memory-conscious, and for some situations this default strategy wastes precious space, so we introduce our own strategy, which increases the space taken by 10%

IMPLEMENTATION DECISIONS:
Template class to work with different containers
