Detailed Description

A collection of useful functionality to order collections of objects.

DESCRIPTION:
This implements some generic commonly occurring ordering functionality. In particular,

  1. Assorted comparison objects which handle derefencing optional and pointer types, and unwrapping reference wrapped types.
  2. Lexicographical comparison for small collections of objects with distinct types (std::lexicographical_compare only supports a single type).
  3. Efficiently, O(N log(N)), simultaneously sorting equal length collections supporting random access iteration using one the collections to to define the order for all.

static

Simultaneously sort multiple vectors using pred order of keys.

This simultaneously sorts a number of vectors based on ordering a collection of keys. For examples, the following code:

std::vector<double> ids{3.1, 2.2, 0.5, 1.5};
std::vector<std::string> names{"a", "b", "c", "d"};
maths::common::COrderings::simultaneousSort(ids, names);
for (std::size_t i = 0; i < 4; ++i) {
std::cout << ids[i] << ' ' << names[i] << std::endl;
}
