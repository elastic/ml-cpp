Detailed Description

Manages accessing the bytes of common::CFloatStorage.

DESCRIPTION:
We pack the row's split indices (four to a float) into the raw bytes of the data frame. This means we only need to lookup aggregate derivative bucket to update in CBoostedTreeLeafNodeStatistics::addRowDerivatives. This manages reading and writing bits to an array of four std::uint8_t types.
