Detailed Description

Collection of set utility functions not provided by the STL.

DESCRIPTION:
This implements various set related functionality which can't be implemented efficiently using the STL. For example, a function for computing set difference in-place, and functions for counting elements in set differences and unions. Common measures of set similarity such as the Jaccard index are also implemented.
