Compute the values to add to the trend and each component.

Parameters
[in]trendThe long term trend prediction.
[in]seasonalThe seasonal components.
[in]calendarThe calendar components.
[in]timeThe time of value to decompose.
[in]deltasThe delta offset to apply to the difference between each component value and its mean, used to minimize slope in the longer periods.
[in,out]decompositionUpdated to contain the value to add to each by component.
[out]predictionsFilled in with the component predictions.
[out]referenceErrorFilled in with the error w.r.t. the trend.
