Detailed Description

A collection of utility functionality for the CModel hierarchy.

DESCRIPTION:
A collection of utility functions primarily intended for use by the CModel class hierarchy.

IMPLEMENTATION DECISIONS:
This class is really just a proxy for a namespace, but a class has been intentionally used to provide a single point for the declaration and definition of utility functions within the model library. As such all member functions should be static and it should be state-less. If your functionality doesn't fit this pattern just make it a nested class.
