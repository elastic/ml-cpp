Detailed Description

Convert a stream of bytes into Base64.

DESCRIPTION:
Class to convert bytes into Base64 using the boost iostreams filter interface.

IMPLEMENTATION DECISIONS:
Uses a boost circular_buffer to cache chunks of up to 4096 bytes from the incoming stream, as Base64 can only be encoded in multiples of 3. This class is templated on the stream output type, which must be compatible with boost::iostreams::write. Stream data is copied from the incoming stream, processed locally, then copied into the output stream; this could be improved, at the expense of simplicity, if it proves to be overly slow.
