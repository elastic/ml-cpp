Detailed Description

template<typename T>
class ml::maths::common::CSampling::CVectorDissimilaritySampler< T >

Perform a weighted sample of stream of vectors of specified cardinality where the probability of sampling each vector is proportional to its average distance from the other sampled vectors.

DESCRIPTION:
Each vector is sampled with probability

Here, is distributed according to the empirical distribution of the selected samples. This means samples already selected repel nearby ones from subsequently being selected which has the effect tending to spread them more regularly.
