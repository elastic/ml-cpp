Detailed Description

Finds the value to add to a set of predicted log-odds which minimises adjusted regularised cross entropy loss w.r.t. the actual categories for incremental training.

DESCRIPTION:
This applies a correction to the loss based on the cross entropy between the new predictions and the predictions of a supplied tree (the one being retrained).
