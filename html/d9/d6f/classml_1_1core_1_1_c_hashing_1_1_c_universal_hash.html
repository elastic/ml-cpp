Detailed Description

Implements universal hashing on integers.

See http://en.wikipedia.org/wiki/Universal_hashing for discussion.

Warning
The hashes use the prime 4294967291 so rely on the fact that integers being hashed are smaller than this. This should be sufficient for our use cases!
