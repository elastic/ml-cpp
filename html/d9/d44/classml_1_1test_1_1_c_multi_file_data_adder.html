Detailed Description

A file based persister for writing Ml models.

DESCRIPTION:
Output file paths are a concatenation of the baseFilename passed to the constructor, a hardcoded "_index" directory, the "id" argument to the persistence method and the file extension passed to the constructor (default '.json').

IMPLEMENTATION DECISIONS:
Only stream-based methods are presented here, as persistence with large models can run to many gigabytes of data

Data is added in multiple streams, each written to a different file. For an option to persist an entire model to a single file, use the CSingleStreamDataAdder class.
