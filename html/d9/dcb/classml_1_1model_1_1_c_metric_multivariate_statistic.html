Detailed Description

template<typename STATISTIC>
class ml::model::CMetricMultivariateStatistic< STATISTIC >

Wraps up one of our basic statistic objects for vector valued quantities.

DESCRIPTION:
This is a wrapper around our basic metric statistic to enable vector valued statistics. In particular, it implements the contract for our basic statistic object for vector valued quantities.

Template Parameters
STATISTIChas the following requirements:
  1. Member function void add(double value, unsigned int count)
  2. Implementations for every function in CMetricStatisticsWrapper
  3. Member operator +=

  5. Supported by core::memory_debug::dynamicSize
  6. Supported by core::memory::dynamicSize
