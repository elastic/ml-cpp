Detailed Description

template<class STATISTIC>
class ml::model::CMetricPartialStatistic< STATISTIC >

A partial metric statistic.

DESCRIPTION:
A partial statistic is composed by the value of the statistic and the mean time of the measurements that compose the statistic. The class groups the two together and provides functions in order to add measurements to the statistic, combine it with others and read its values.

Template Parameters
STATISTIChas the following requirements:
  1. Member function void add(double value, unsigned int count)
  2. Implementations for every function in CMetricStatisticsWrapper
  3. Member operator +=
  4. Supported by maths::common::CChecksum::calculate
  5. Supported by core::memory_debug::dynamicSize
  6. Supported by core::memory::dynamicSize
  7. Have overload of operator<<
