Detailed Description

Aggregates the probabilities up a collection hierarchical results.

DESCRIPTION:
This can be used to compute the aggregate probabilities for the internal nodes of a hierarchical results object in a bottom up breadth first pass.

This uses different aggregation styles for aggregating partitions, people in a population and collections of individual results in system change analysis. Special logic is used for named people, i.e. aggregations of multiple compatible simple searches.
