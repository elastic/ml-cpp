Detailed Description

template<typename T, typename CONTAINER, typename LESS>
class ml::maths::common::CBasicStatistics::COrderStatisticsImpl< T, CONTAINER, LESS >

Implementation of an accumulator class for order statistics.

DESCRIPTION:
This implements the underlying algorithm for determining the first n order statistics online.

IMPLEMENTATION DECISIONS:
This maintains the statistics in a heap for worst case complexity and typical complexity (by checking against the maximum value) for .

The container is supplied as a template argument so that a fixed size array can be used for the case n is small. Similarly the less function is supplied so that T can be any type which supports a partial ordering. (T must also have a default constructor.)
