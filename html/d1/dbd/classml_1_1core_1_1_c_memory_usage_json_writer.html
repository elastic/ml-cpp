Detailed Description

A lightweight wrapper around CStreamWriter to be used by CMemoryUsage to format DebugMemoryUsage info in a JSON string.

DESCRIPTION:
This class is used by CMemoryUsage to format DebugMemoryUsage info in a JSON string. A stream is passed into the class at construction and this is then filled with JSON objects and arrays. Finalise should be called to flush the stream before downstream use.
