Detailed Description

A node of a regression tree.

DESCRIPTION:
This defines a tree structure on a vector of nodes (maintaining the parent child relationships as indexes into the vector). It holds the (binary) splitting criterion (feature and value) and the tree's prediction at each leaf. The intervals are open above so the left node contains feature vectors for which the feature value is strictly less than the split value.

During training row masks are maintained for each node (so the data can be efficiently traversed). This supports extracting the left and right child node bit masks from the node's bit mask.
