Detailed Description

template<typename PRIOR>
class ml::CPriorTestInterfaceMixin< PRIOR >

A mix in of test interface which brings the necessary functions into scope and implements value semantics.

IMPLMENTATION:
This is variant of the curiously recurring template pattern to mix in some interface for test purposes only.

Note that this also uses double inheritance, contravening the coding standards, because it's the cleanest way to implement this functionality. DON'T use this elsewhere.
