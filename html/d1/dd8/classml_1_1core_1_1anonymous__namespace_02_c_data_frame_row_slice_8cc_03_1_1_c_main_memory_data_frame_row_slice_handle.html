Detailed Description

A handle for reading CRawDataFrameRowSlice objects.

DESCRIPTION:
This stores a reference to the underlying memory. This is primarily intended to stop duplication of the underlying data frame state. Together with our memory mapped vector type this means algorithms on top of a raw data frame can work entirely in terms of the raw values stored in the data frame.
