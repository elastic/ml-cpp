Detailed Description

Similar idea to boost::noncopyable, but for instantiation.

DESCRIPTION:
Classes for which instantiation is not allowed, i.e. those that have only static methods, should inherit privately from this class.

IMPLEMENTATION DECISIONS:
The class is (seemingly pointlessly) exported from the DLL to prevent Visual C++ warning C4275.
