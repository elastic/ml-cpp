Detailed Description

The most basic individual model interface.

DESCRIPTION:
This implements or stubs out the common portion of the CAnomalyDetectorModel interface for all individual models. It holds the maths:: and objects which are used to describe the R.V.s that we use to model individual time series' features.

IMPLEMENTATION DECISIONS:
This gathers up the implementation which can be shared by all event rate and metric individual times series models to avoid unnecessary code duplication.

It assumes data are supplied in time order since this means minimal state can be maintained.
