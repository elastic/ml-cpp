enum ml::counter_t::ECounterTypes

The enum values must be explicitly assigned & names should have a meaningful prefix to effectively namespace counters New counters may be added anywhere before E_LastEnumCounter and must be grouped by namespace prefix. The E_LastEnumCounter value must be bumped for every new enum added. Don't forget to also add a description of the new enum value to m_CounterDefinitions.
