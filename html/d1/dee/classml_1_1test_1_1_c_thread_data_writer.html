Detailed Description

Writes to a file in a thread.

DESCRIPTION:
When the thread is started it waits a short time, then writes data of a specified size to a file. The data consists of a single specified character repeated.

IMPLEMENTATION DECISIONS:
Using core::CThread rather than std::thread to minimise risk as it was factored out of existing code using core::CThread.
