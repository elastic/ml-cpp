Detailed Description

A cache of the probability calculation to use in cases that many probabilities are being computed from the same model.

DESCRIPTION:
This caches the probabilities for each feature and attribute since they can be expensive to compute and for large populations we can repeatedly calculate probabilities for the same model and similar parameters.

This bounds the maximum relative error it'll introduce by only interpolating an interval if the difference in the probability at its end points satisfy .
