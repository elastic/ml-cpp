Detailed Description

A very simple model for counting events in the sampled bucket.

DESCRIPTION:
This performs no analysis and is used by the simple count detector to get the sampling bucket event rate.

IMPLEMENTATION DECISIONS:
This is lightweight as possible and in particular avoids creating any time series models. This avoids a potential pathology where memory limiting can cause us to stop getting counts and also makes interpreting the maths library logging easier.
