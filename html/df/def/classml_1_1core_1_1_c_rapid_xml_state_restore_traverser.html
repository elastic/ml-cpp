Detailed Description

For restoring state in XML format.

DESCRIPTION:
Concrete implementation of the CStateRestoreTraverser interface that restores state in XML format.

IMPLEMENTATION DECISIONS:
Directly uses RapidXml to avoid the inefficiency of an intermediate node hierarchy.

Stores a const reference to an XML parser, which must not be modified externally during the lifetime of any object of this class that references it.

Does NOT support CDATA in state XML - any CDATA content is currently ignored.
