Detailed Description

Interface for simple output of homogenous key/value pairs.

DESCRIPTION:
Interface to the various ways of outputting key/value data. Usually the output will be to a C++ stream after formatting into NDJSON or CSV.

IMPLEMENTATION DECISIONS:
Since the field names should never change for a given job, it is possible to massively improve efficiency by pre-computing the hashes for the strings that hold the field names. The nested CPreComputedHash class and TPreComputedHashVec typedef can be used to implement this.
