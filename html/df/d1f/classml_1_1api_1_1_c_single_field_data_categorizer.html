Detailed Description

Wraps a categorizer, mapping local IDs to global IDs.

DESCRIPTION:
Exposes the functionality of the model library data categorizer classes, but with local category IDs mapped to the corresponding global category IDs.

IMPLEMENTATION DECISIONS:
The persistence for this class is unusual in that it is not persisted within a new sub-level in the state. The reason for this is that the objects it contains used to be persisted by the CFieldDataCategorizer when there was only one low level categorizer. An extra level cannot be introduced now, as that would break backwards compatibility of the state.
