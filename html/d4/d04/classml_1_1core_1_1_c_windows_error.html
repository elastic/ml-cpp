Detailed Description

Encapsulate interpretation of Windows function errors.

DESCRIPTION:
Encapsulate interpretation of Windows function errors. On Windows most API functions set an error number that is retrieved using the GetLastError() function. The code required to translate this to a human readable string is quite verbose, so this class encapsulates it.

IMPLEMENTATION DECISIONS:
The default constructor initialises an object using the last error that occurred. However, it's also possible to explicitly initialise an object with a specific error code.
