◆ decreaseMargin()

void ml::model::CResourceMonitor::decreaseMargin ( core_t::TTime elapsedTime)

Decrease the margin on the memory limit.

We start off applying a 'safety' margin to the memory limit because it is difficult to accurately estimate the long term memory usage at this point. This safety margin is gradually decreased over time by calling this once per bucket processed until the initially requested memory limit is reached.
