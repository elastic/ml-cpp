Detailed Description

Abstract interface for classes that process data records.

DESCRIPTION:
Classes that process data records must implement this interface so that they can fit into the CCmdSkeleton framework.

std::string ml::api::CDataProcessor::m_TimeFieldName

Name of field holding the time. An empty string, indicates the input contains no timestamp. This may not be valid for some data processors, in which case the derived class must validate this field.
