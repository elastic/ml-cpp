Detailed Description

Time series data gathering interface.

DESCRIPTION:
This defines the interface to gather time-specific data for one or more time series.

This is subclassed by Metric and EventRate implementations.

IMPLEMENTATION DECISIONS:
This functionality has been separated from the CDataGatherer in order to allow the CDataGatherer to support multiple overlapping buckets and buckets with different time spans. However, the overlapping feature has been removed but this class is kept to avoid BWC issues.
