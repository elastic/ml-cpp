The influence calculation for features using computeSample to get the statistics and computeInfluence to compute the influences from the corresponding probabilities.

Parameters
[in]computeInfluencedParamsAndValueThe function to compute the influenced feature value for which to compute the probability.
[in]computeInfluenceThe function to compute influence.
[in]modelThe model to use to compute the probability.
[in]elapsedTimeThe time elapsed since the model was created.
[in]computeProbabilityParamsThe parameters need to compute the probability.
[in]timeThe time of value.
[in]valueThe influenced feature value.
[in]countThe measurement count in value.
[in]influencerNameThe name of the influencer field.
[in]influencerValuesThe feature values for the intersection of the records in value with distinct values of influenceName.
[in]cutoffThe value at which there is no influence.
[in]includeCutoffIf true then add in values for influences less than the cutoff with estimated influence.
[out]resultFilled in with the influences of value.
