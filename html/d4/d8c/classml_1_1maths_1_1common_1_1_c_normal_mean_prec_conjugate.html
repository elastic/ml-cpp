Detailed Description

A conjugate prior distribution for a normal variable.

DESCRIPTION:
Our normal variable Y is assumed to have unknown mean and precision.

The parameters of Y are modeled as a joint gamma-normal distribution (which is the conjugate prior for a normal with unknown mean and precision).

All prior distributions implement a process whereby they relax back to the non-informative over some period without update (see propagateForwardsByTime). The rate at which they relax is controlled by the decay factor supplied to the constructor.

IMPLEMENTATION DECISIONS:
All priors are derived from CPrior which defines the contract that is used by composite priors. This allows us to select the most appropriate model for the data when using one-of-n composition (see COneOfNPrior) or model data with multiple modes when using multi-modal composition (see CMultimodalPrior). From a design point of view this is the composite pattern.
