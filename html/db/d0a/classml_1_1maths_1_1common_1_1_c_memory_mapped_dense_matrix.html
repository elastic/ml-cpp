Detailed Description

template<typename SCALAR, Eigen::AlignmentType ALIGNMENT>
class ml::maths::common::CMemoryMappedDenseMatrix< SCALAR, ALIGNMENT >

Decorates an Eigen::Map of a dense matrix with some useful methods and changes default copy semantics to shallow copy.

IMPLEMENTATION DECISIONS:
This effectively acts like a std::reference_wrapper of an Eigen::Map for an Eigen matrix. In particular, all copying is shallow unlike Eigen::Map that acts directly on the referenced memory. This is to match the behaviour of CMemoryMappedDenseVector.

See also
CMemoryMappedDenseVector for more information.
