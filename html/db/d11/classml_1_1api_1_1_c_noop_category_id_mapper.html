Detailed Description

Category ID mapper for use when local and global category IDs are the same.

DESCRIPTION:
All mappings are no-ops and the categorizer key is the empty string.

IMPLEMENTATION DECISIONS:
This mapper is designed for the case where there is a single categorizer, which includes all jobs created before 7.9.
