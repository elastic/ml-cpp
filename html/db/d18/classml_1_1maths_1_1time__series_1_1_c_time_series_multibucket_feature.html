Detailed Description

template<typename>
class ml::maths::time_series::CTimeSeriesMultibucketFeature< typename >

Defines an interface for features of multiple time series values.

DESCRIPTION:
The intention is to provide useful features for performing anomaly detection in addition to the current time series value. Specifically, unusual values of properties of extended intervals of a time series are often the most interesting events from a user's perspective.
