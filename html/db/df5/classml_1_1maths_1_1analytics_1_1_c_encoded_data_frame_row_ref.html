Detailed Description

A wrapper of a data frame row reference which handles encoding categories.

DESCRIPTION:
This implements a subset of the core::CDataFrame::TRowRef interface dealing with the extra dimensions introduced by one-hot encoded categories, extracting the mean target variable value for other categories and so on. The intention is it provides a drop in replacement for core::CDataFrame::TRowRef when a data frame contains categorical columns.
