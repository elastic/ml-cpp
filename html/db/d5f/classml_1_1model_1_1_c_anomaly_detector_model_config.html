Detailed Description

Responsible for configuring anomaly detection models.

DESCRIPTION:
Responsible for configuring classes for performing anomaly detection. It also defines all parameter defaults.

IMPLEMENTATION DECISIONS:
This wraps up the configuration of anomaly detection to encapsulate the details from calling code. It is anticipated that:

  1. Some of this information will be exposed to the user via a configuration file,
  2. Some may be calculated from data characteristics and so on.
