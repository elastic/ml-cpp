Detailed Description

Maintains a collection of statistics about a leaf of the regression tree as it is built.

DESCRIPTION:\N The regression tree is grown top down by greedily selecting the split with the maximum gain (in the loss). This finds and scores the maximum gain split of a single leaf of the tree.
