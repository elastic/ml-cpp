Detailed Description

Event rate data gathering class.

DESCRIPTION:
This performs all pre-processing of the data, which we use in order to model the event rate in an arbitrary time series.

See also
CDataGatherer.

◆ fieldsOfInterest()

const CEventRateBucketGatherer::TStrVec & ml::model::CEventRateBucketGatherer::fieldsOfInterest ( ) const
overridevirtual

Get the fields for which to gather data.

For individual searches this gets the field which defines the categories whose counts are being analyzed. For population searches this gets the fields identifying the people and person attributes which are being analyzed. An empty string acts like a wild card and matches all records.
