Detailed Description

template<typename POINT, EInfoCriterionType TYPE>
class ml::maths::common::CGaussianInfoCriterion< POINT, TYPE >

Computes the information content of a collection of point clouds under the assumption that they are distributed as a weighted sum of Gaussians.

DESCRIPTION:
This places no restriction on the covariance matrix in particular it is assumed to have parameters. For more details on the information criteria see CSphericalGaussianInfoCriterion.
