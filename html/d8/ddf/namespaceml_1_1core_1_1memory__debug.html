Detailed Description

Core memory debug usage template class.

DESCRIPTION:
Core memory debug usage template class. Provides an extension to the CMemory class for creating a detailed breakdown of memory used by classes and containers, utilising the CMemoryUsage class.

ML classes can declare a public member function:

void debugMemoryUsage(const CMemoryUsage::TMemoryUsagePtr&) const;
ml::core::CMemoryUsage::TMemoryUsagePtr
std::shared_ptr< CMemoryUsage > TMemoryUsagePtr
Definition CMemoryUsage.h:53

which should call memory_debug::dynamicSize("t_name", t, memUsagePtr) on all its dynamic members.
