Detailed Description

Performs encoding of the categorical columns in a data frame.

DESCRIPTION:
We consider:

  1. One-hot encoding for the categories which carry the most information (because this is lossless)
  2. Mean target encoding for categories where we have a reasonable sample in the training data and
  3. Frequency encoding

We simultaneously select the categorical fields to encode, the encoding scheme and metric features to use by minimising a MICe based measure of their relevency for prediction and redundancy. The total number of features we select is based on the quantity of training data.
