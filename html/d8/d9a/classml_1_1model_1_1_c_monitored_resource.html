Detailed Description

Interface for resource monitoring.

DESCRIPTION:
Resources that the CResourceMonitor class monitors must derive from this interface.

IMPLEMENTATION DECISIONS:
Defaults to the assumption that monitored resources do not support pruning. Methods related to pruning have default implementations consistent with this assumption.
