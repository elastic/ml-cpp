Detailed Description

template<typename POINT>
class ml::maths::common::CKMeans< POINT >::CClosestPointsCollector

Extracts the closest points to each centre from a k-d tree in a single traversal.

DESCRIPTION:
This is used in a post-order depth first traversal of the k-d tree of points to extract the closest points to each centre supplied to the constructor.
