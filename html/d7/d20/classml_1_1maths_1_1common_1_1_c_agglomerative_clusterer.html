Detailed Description

Implements optimum runtime agglomerative clustering for arbitrary distance matrices.

DESCRIPTION:
Agglomerative clustering builds a binary tree bottom up on a set of points. At each stage it merges the closest pair of clusters subject to a specified distance function. This implements a few choices, comprising single, complete, average, weighted link and ward, for which an optimum algorithm is known.
