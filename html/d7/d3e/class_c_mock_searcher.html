Detailed Description

Mock searcher for unit testing.

DESCRIPTION:
The CSearcher class can search for data in a live server instance, but for unit testing it's desirable to mock this class.

IMPLEMENTATION DECISIONS:
The mock searcher's returns all events added to a mock data adder that appear to be for the searched index. The actual search string is NOT properly applied. This is OK for the current scope of the unit testing.
