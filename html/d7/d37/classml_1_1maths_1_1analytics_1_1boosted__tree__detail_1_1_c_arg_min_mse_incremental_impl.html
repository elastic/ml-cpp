Detailed Description

Finds the value to add to a set of predictions which minimises the adjusted regularized MSE difference w.r.t. the actual values for incremental training.

DESCRIPTION:
This applies a correction to the loss based on the difference from the predictions of a supplied tree (the one being retrained).
