Detailed Description

Helper class to register a crashhandler and getting better traces.

DESCRIPTION:
Last line of defense when the autodetect process crashes. Tries to get out useful information on a SIGSEGV and logs them.

IMPLEMENTATION DECISIONS:
The basic implementation does nothing. Platform-specific implementations exist for platforms where we have decided to do something.

Currently the only platform-specific implementation is for Linux. This might work for Mac as well.

On Linux the handler returns signal number, code, errno (meaning can be found in siginfo.h), address, library, base and normalized_address. To turn that into a line number use:

addr2line -e library normalized_address (library can be the symbol file)
