Detailed Description

Collects up to a configurable number of distinct examples per category.

IMPLEMENTATION DECISIONS:
The examples are stored sorted in a boost flat_set. The flat_set was preferred due to the contiguous storage which in combination with the small number of expected examples should be more performant than a traditional set.
