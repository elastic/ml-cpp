Detailed Description

Equalizers a set of detector probabilities at a result node.

DESCRIPTION:
This maintains a quantile sketch of a set of minus log probabilities for each detector. A corrected probability is obtained by converting raw probabilities to a rank and then reading off median probability for that rank over all detectors.
