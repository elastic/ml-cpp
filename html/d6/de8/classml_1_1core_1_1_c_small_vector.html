Detailed Description

template<typename, std::size_t>
class ml::core::CSmallVector< typename, size_t >

This inherits from boost::container::small_vector.

DESCRIPTION:
The reasons for this class are largely historical in that we didn't have boost::container::small_vector in the version of boost we were using when this was originally implemented. However, a lot of our code now uses this class and it is convenient to provide some non-standard extensions.

IMPLEMENTATION DECISIONS:
Inherits from boost::container::small_vector.

Template Parameters
TThe element type.
NThe maximum number of elements which are stored on the stack.
