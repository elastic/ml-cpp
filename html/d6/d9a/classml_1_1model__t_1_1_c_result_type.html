Detailed Description

Describes the result type.

DESCRIPTION:
The types of the results:

  1. Interim: if the bucket isn't complete.
  2. Final: if the bucket is complete.
  3. Unconditional: if a result corresponds to a feature value which is unusual in its own right.
  4. Conditional: if a result corresponds to a feature value which is unusual only after conditioning on some other.

IMPLEMENTATION DECISIONS:
Uses 1-of-n bitwise encoding of the different result binary descriptors so the size is just that of an unsigned int.
