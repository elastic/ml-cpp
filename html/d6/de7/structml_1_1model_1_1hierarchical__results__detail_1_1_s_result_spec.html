Detailed Description

The data fully describing a result node.

DESCRIPTION:
A result is fully described by the complete set of field names and values in a simple search. If one or more of these unset then the node corresponds to come aggregation of simple results.

A simple search corresponds to a single clause in the search config file. It corresponds to the following command line syntax: 

  autodetect <function>[(X)] [by Y] [over Z]

So, examples include

  1. autodetect count
  2. autodetect count by status
  3. autodetect sum(bytes) over host
  4. autodetect rare by uri_path over clientip
  5. and so on.
