Detailed Description

Write output data in JSON format, one document per line.

DESCRIPTION:
This class writes every result passed to it as a separate JSON document. Each document is restricted to a single line so that whatever process consumes the output can determine where one document ends and the next starts.

IMPLEMENTATION DECISIONS:
Using boost::json to do the heavy lifting.
