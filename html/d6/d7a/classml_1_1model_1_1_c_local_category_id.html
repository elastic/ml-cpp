Detailed Description

Adds type safety to a local category ID.

DESCRIPTION:
Local category IDs are the IDs used by the categorization classes in the model library. Each model library categorizer will have local category IDs running from 1 to the number of categories found.

IMPLEMENTATION DECISIONS:
This is basically a wrapper around an int, but by using a dedicated class there is less risk of accidentally confusing other ints.
