Detailed Description

Manages the aggregation of probabilities.

DESCRIPTION:
This allows one to register either one of or both the joint probability and extreme aggregation styles. The resulting aggregate probability is the minimum of the aggregates of the probabilities added so far for any of the registered aggregation styles.
