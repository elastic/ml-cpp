Detailed Description

Portable wrapper for the uname() function.

DESCRIPTION:
Portable wrapper for the uname() function.

IMPLEMENTATION DECISIONS:
This has been broken into a class of its own because Windows does not implement Unix's uname() function.

Each member of the utsname struct should be implemented as a separate method, as usually only one is required.
