Detailed Description

Interface for classes that convert a raw event string to a category.

DESCRIPTION:
Abstract interface for classes that convert a raw event string to a category.

IMPLEMENTATION DECISIONS:
At the time of writing, only the token list data categorizer implements this interface. However, it is not hard to imagine a time when there are specialist data categorizers for XML, JSON or delimited files, so it is good to have an abstract interface that they can all use.
