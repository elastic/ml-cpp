Detailed Description

Manages a greedy search for the minimum redundancy maximum relevancy feature set.

DESCRIPTION:
Implements a greedy search to approximately solve the optimization problem

This trades redundancy of information in the feature set as a whole and relevance of each individual feature when deciding which to include. We extend the basic measure by including a non-negative redundancy weight which controls the priority of minimizing redundancy vs maximizing relevancy. For more information see https://en.wikipedia.org/wiki/Feature_selection#Minimum-redundancy-maximum-relevance_(mRMR)_feature_selection and references therein.
