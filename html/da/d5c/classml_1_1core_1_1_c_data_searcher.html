Detailed Description

Interface class for retrieving data by running a search.

DESCRIPTION:
Interface for searching for data in some sort of data store. Derived classes will supply the details of how to search a specific type of data store.

IMPLEMENTATION DECISIONS:
Abstract class.

The result of a successful search is a std::istream.
