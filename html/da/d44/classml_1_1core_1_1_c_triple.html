Detailed Description

template<typename T1, typename T2, typename T3>
class ml::core::CTriple< T1, T2, T3 >

A tuple with three elements.

IMPLEMENTATION DECISIONS:
This implements a lightweight version of boost::tuple with syntax similar to std::pair for the case of three elements.

It also implements hash_value which isn't implemented by boost::tuple meaning it can be used as a boost::unordered_map key provided the underlying types can be hashed using a boost::hasher.
