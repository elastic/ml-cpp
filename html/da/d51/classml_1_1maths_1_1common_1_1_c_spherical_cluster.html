Detailed Description

template<typename POINT>
class ml::maths::common::CSphericalCluster< POINT >

Defines the type of a spherical cluster using the specified point type.

This annotates the POINT vector corresponding to the centre of the cluster with the count of points in the cluster and the spherically symmetric variance of those points.
