Detailed Description

Convert CSeasonalTime sub-classes to/from text representations.

DESCRIPTION:
Encapsulate the conversion of arbitrary CSeasonalTime sub-classes to/from textual state. In particular, the field name associated with each type of CSeasonalTime is then in one file.
