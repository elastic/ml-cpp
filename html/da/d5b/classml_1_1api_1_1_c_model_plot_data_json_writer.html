Detailed Description

Write visualisation data as a JSON document.

DESCRIPTION:
JSON is either written to the output stream or an internal string stream. The various writeXXX functions convert the arguments to a JSON doc and write to the stream then flush the stream. If the object is constructed without an outputstream then the doc can be read via the internalString function.

IMPLEMENTATION DECISIONS:
The stream is flushed after at the end of each of the public write.... functions.
