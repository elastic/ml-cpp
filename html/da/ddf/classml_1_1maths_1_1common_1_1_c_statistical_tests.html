static

A two sample Kolmogorov-Smirnov test.

This computes the test significance for rejecting the null hypothesis that x and y are samples from the same distribution. The smaller the significance the more likely that x and y come from different distributions.

Note
This is based on the implementation in Numerical Recipes in C.
