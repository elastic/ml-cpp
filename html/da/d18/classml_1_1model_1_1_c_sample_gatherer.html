Detailed Description

template<typename STATISTIC, model_t::EFeature FEATURE>
class ml::model::CSampleGatherer< STATISTIC, FEATURE >

Metric statistic gatherer.

DESCRIPTION:
Wraps up the functionality to sample a statistic of a fixed number of metric values, which is supplied to the add function, for a latency window.

This also computes the statistic value of all metric values and for each influencing field values for every bucketing interval in the latency window.

Template Parameters
STATISTICThis must satisfy the requirements imposed by CMetricPartialStatistic.
